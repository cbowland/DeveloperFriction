# Developer Friction Survey

The Developer Friction Survey is a single-page application that collects feedback from developers across three key friction areas and visualizes aggregated results in real time.

## Overview

The app presents three survey questions, each with three radio button options ranging from low to high friction. After a developer submits their response, the app displays a grouped bar chart showing how all respondents have answered so far.

This tool is designed to give platform teams a quick, lightweight way to measure developer experience without requiring a full survey platform.

## Key Features

- Three targeted questions covering environment setup, CI/CD, and component reuse
- Real-time aggregated results displayed as a grouped bar chart
- No database required -- responses are stored in memory
- Single container deployment on OpenShift
- Registered and visible in Red Hat Developer Hub
